Returns & Exchanges

Easy returns within 30 days

30-Day Return Policy

We want you to love your purchase. If you're not completely satisfied, return unworn items with tags within 30 days for a full refund.

Free Returns

All returns are free. Request a return label through your account and we'll email you a prepaid shipping label.

Exchanges

Need a different size or color? Exchanges are free and priority processed.

How to Return

  1. Log into your account and go to order history
  2. Select items to return and choose reason
  3. Print prepaid return label
  4. Package items with original tags
  5. Drop off at any UPS/USPS location
  6. Refund processed within 5-7 business days
